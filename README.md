# 🎓 You VS Teacher - Quiz Game
 
Un juego educativo interactivo construido con **Pygame** que desafía a los jugadores a responder preguntas de múltiple opción en categorías de Matemáticas, Geografía e Inglés. Con soporte de **síntesis de voz (TTS)**, diseño accesible y seguimiento de estadísticas.
 
---
 
## ✨ Características Principales
 
- **🎮 Gameplay Dinámico**: 
  - Preguntas aleatorias de 3 categorías diferentes
  - Interfaz moderna y accesible con alto contraste
  - Feedback visual inmediato
- **🔊 Accesibilidad Avanzada**:
  - Síntesis de voz (TTS) integrada con `pyttsx3`
  - Lectura automática de preguntas y opciones
  - Control por teclado y ratón
  - Atajos de teclado (A, B, C, D o 1, 2, 3, 4)
- **🌍 Multiidioma**:
  - Soporte completo para Español e Inglés
  - Cambio de idioma en tiempo real
  - Interfaz bilingüe
