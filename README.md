# 🎓 You VS Teacher | Tu VS Professor

![Python Version](https://img.shields.io/badge/python-3.8%2B-blue)
![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20Linux%20%7C%20macOS-lightgrey)

**You VS Teacher** és un joc de trivia interactiu basat en consola i desenvolupat en Python. Desafia els teus coneixements en matemàtiques, geografia i anglès mentre competeixes contra el "professor". 

El projecte compta amb **suport natiu de text a veu (TTS)** i interfícies localitzades de forma dinàmica, fet que el converteix en una eina altament accessible per a persones amb discapacitat visual o estudiants d'idiomes.

---

## ✨ Característiques Principals

*   **🌐 100% Bilingüe:** Suport complet per a Anglès (`en`) i Espanyol (`es`), canviant dinàmicament tant el text a la consola com la veu del sistema.
*   **🔊 Accessibilitat Nativa (Mode Cec):** Dissenyat amb un flux on la pantalla es renderitza primer i el motor TTS llegeix les preguntes i opcions de forma ordenada, evitant interrupcions molestes.
*   **📚 Banc de Preguntes Multidisciplinari:** 
    *   🧮 **Math / Matemàtiques:** Operacions aritmètiques i lògica bàsica.
    *   🌍 **Geography / Geografia:** Capitals, continents i dades globals.
    *   🇬🇧 **English / Anglès:** Gramàtica, temps verbals i vocabulari.
